# OrangeHRM Selenium Automation Framework

Automation framework developed using:

- Java 21
- Selenium 4
- TestNG
- Maven
- Page Object Model (POM)
- Extent Reports

## Features

- Login Automation
- Dashboard Validation
- Leave Module Validation
- Screenshot Capture on Failure
- Extent Reporting
- Reusable Utilities

## Project Structure

src/main/java
- pages
- utils

src/test/java
- tests

## Tools Used

- Eclipse IDE
- Maven
- Git
- GitHub
- Chrome Browser
